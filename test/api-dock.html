Dock Icons

List of test devices

This page tests adding and removing dock icons.

Adding onReady

ActionExpected result(s)Works?
Load the player Firefox, IE8: Player loads in Flash. Otherwise, player loads in HTML5 mode
Begin playing Player controls become visible, including a dock icon.
Mouse over dock icon Label should appear underneath the dock icon.
Click dock icon An alert should appear saying "Button clicked".

Adding Later

ActionExpected result(s)Works?
Load the player Firefox, IE8: Player loads in Flash. Otherwise, player loads in HTML5 mode
Begin playing Player controls become visible, no dock icon.
Click the "Add a Button" button, then mouse over player Dock icon is now visible.

Removing

ActionExpected result(s)Works?
Load the player Firefox, IE8: Player loads in Flash. Otherwise, player loads in HTML5 mode
Begin playing Player controls become visible, including a dock icon.
Click "Remove the Button" button, then mouse over the player The dock icon is no longer visible.
