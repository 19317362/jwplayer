Getting Started

Getting JW Player up and running is now easier than ever. Only a few lines of HTML code is needed to embed a video that works on any desktop, tablet and phone. Since the player is now CDN hosted, you don't even have to upload the player to your server anymore.

Uploading the Video

In order to embed the player, you first need a video file uploaded to your server. Ensure it is in the MP4 format, since that can be played on any browser and device (including iPad, iPhone and Android).

  • If your video is not in the MP4 format, you can convert it using a free tool like Handbrake. See our Video Encoding guide for more info.
  • If you have no video at hand, feel free to use this short clip for your testing.
  • If you have no web server for uploading videos, check out our video platform. It offers you a free GB of storage, plus it automatically converts your video to MP4.

Additionally, you should upload a poster image to your server. This image (in JPG format) is displayed before the video starts. An interesting frame from the video usually works well to draw attention and entices users to click play. Here is an image for our short test clip.

Embedding the Player

Since JW Player is hosted on our content delivery network (CDN), there is no need to install and upload the player itself to your site. Therefore, embedding the JW Player is a simple, 3-step process. Open your web page in an HTML editor and:

  1. Insert the JW Player script from our CDN into the head of your page: 
    <script type="text/javascript" src="http://jwpcdn.com/player.js" ></script>
  2. Insert a <div> element with a unique id at the location you want the player to appear: 
    <div id="container">Loading the player ...</div>
  3. Insert a script that invokes the JW Player setup() call on your <div>:
    4. <script type="text/javascript">
    jwplayer("container").setup({
        file: "/uploads/example.mp4",
        height: 270,
        image: "/uploads/example.jpg",
        width: 480
    });
</script>

In the setup() block, file and image should be the URLs to your video file and poster images. Their paths should be relative to the HTML page (just like when embedding an image using <img>). The height and width are used to set the dimensions of the player. Feel free to change these to values that make the player nicely fit your page.
